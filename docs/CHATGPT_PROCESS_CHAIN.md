1.1 熟铁（回顾）

块炼直还（C）

需：块炼炉 + 手/足风箱 + 手锻工具组

配方A（还原）：铁矿×10 + 木炭×20 + 石灰石×10 → 海绵铁×10 + 炉渣×10

配方B（整块）：海绵铁×10（热锻）→ 熟铁×12

矿修正：赤铁 +15%、磁铁 +10%、褐/纤 −10%

竖炉/早高炉（M）

需：竖炉/早高炉 + 水力鼓风 + 手/水锻

配方（替换）：铁矿×10 + 木炭×30 + 石灰石×10 → 生铁×22 + 炉渣×10

细炼：生铁×20 + 木炭×10 → 熟铁×15 + 炉渣×5

有洗涤坑：得稀硫酸水×2

高炉堆（R）

需：高炉堆 + 水力鼓风 + 水力锻锤

配方（替换）：铁矿×10 + 木炭×28 + 石灰石×10 → 生铁×24 + 炉渣×10

细炼：生铁×20 + 木炭×8 → 熟铁×16 + 炉渣×4

有淋洗塔：得稀硫酸水×3

炼钢火床（M）

需：炼钢火床（M） + 水力鼓风

改良炼钢火床（R）

需：改良炼钢火床（R） + 水力鼓风

2. 铜线（含青铜与黄铜/卡拉铭）
2.1 炉体与附件升级

坩埚炉（C）：碳酸盐铜（孔雀石/蓝铜矿）直熔；附件：坩埚组、钳具、浇注模

焙烧床（M） ➝ 高草帽焙烧床（R）：硫化铜预焙；R 型烟道抽力更稳

反射炉（M） ➝ 大型反射炉（R）：更大的炉膛与炉门、挡焰墙与风道优化、独立出渣口

密封坩埚台（C→R 通用）：黄铜水泥法专用（铜 + 卡拉铭矿〔菱锌矿/异极矿〕）

2.2 配方

碳酸盐铜（C，坩埚直熔）

孔雀石/蓝铜矿×10 + 木炭×15 + 石灰石×5 → 铜锭×10 + 炉渣×5

矿修正：孔雀石/蓝铜矿 +10%

硫化铜（M/R，焙烧+反射炉）

焙烧：黄铜矿/辉铜矿×10 → 焙烧料×10 + SO₂（M=洗涤坑得稀硫酸水×2；R=淋洗塔得×3）

反射炉：焙烧料×10 + 木炭×12 + 石灰石×6 → 粗铜×10 + 炉渣×6 → 火法精炼 → 铜锭×10

矿修正：黄铜矿 −10%、辉铜矿 −5%

青铜（Cu–Sn，锡石为辅料直接合金）

坩埚合金（C）：铜锭×9 + 锡石×1 + 木炭×2 → 青铜×10（已含约8%熔损）

炉内合金（M/R，反射炉）：粗铜×9 + 锡石×1 + 木炭×2 → 青铜×10（约5%熔损）

黄铜（卡拉铭水泥法，密封坩埚 C→R 通用）

铜锭×20 + 卡拉铭矿×10（菱锌矿/异极矿） + 木炭×10（密封） → 黄铜×20 + 炉渣×5

说明：维持历史水泥法（前锌蒸馏），出料需二次熔炼/锻打均匀

3. 铅/银线（含杯炼与液熔）
3.1 炉体与附件升级

铅熔炉（C→R）：从坩埚/浅炉升级到砖砌短身炉；可接助熔翻渣口

杯炼炉床 + 骨灰杯皿（C→R）：热风/反射增强，M/R可加杯皿回收工位

液熔/赛格炉（R）：专门的铜+铅析银结构；配倾析沟/冷却台/银富集铅锭模

3.2 配方

铅冶（C→R）

方铅矿×10 + 木炭×15 + 石灰石×5 → 粗铅×12 + 炉渣×6 + SO₂（M=洗涤坑得稀硫酸水×2；R=淋洗塔得×3）

杯炼（C→R）（需：杯炼炉床 + 骨灰杯皿）

粗铅×10（含银） → 银×0.3 + 灰铅（PbO）×9.5

灰铅回收

灰铅×9.5 + 木炭×3 → 铅×9 + 炉渣×1

液熔/赛格（R，可选更高回收）（需：赛格炉）

含银铜×10 + 铅×10 → 富银铅×10 + 去银铜×10

后续：富银铅 → 杯炼 → 银产量提升（+10%～+20%）；去银铜回炉成铜锭

4. 朱砂线（Hg）
4.1 炉体与附件升级

盘式焙烧 + 水冷冷凝（C）：铁盘/泥封罩，接水冷凝器

砖砌焙烧炉 + 串联冷凝罐（M）：炉室更稳、冷凝损耗更低

闭式汞蒸馏器（R）：密封坩埚/蛇管冷凝/回收瓶，配抽力烟道与滴汞收集盘

4.2 配方

C：朱砂×10 → 汞×10 + SO₂（仅排放）

M：朱砂×10 → 汞×11 + SO₂（有洗涤坑→稀硫酸水×3）

R：朱砂×10 → 汞×12 + SO₂（有淋洗塔→稀硫酸水×4）

风险：汞蒸气（R 型风险最低；未密封事故/污染风险↑）

5. 金线（沿用铜坩埚附件）

金矿/金砂×10 + 木炭×5 + 助熔（硼砂/石灰）×5 → 金锭×10 + 炉渣×5

若含硫/砷夹杂，M/R 可先用焙烧床预焙（接洗涤装置产稀硫酸水×1–2）

质量（Q）：所有新物品用统一质量阶（如 Q0 普通 / Q1 精良 / Q2 大师 / Q3 典藏）。

装饰动作：只能提升质量，不创建“美观”。失败仅损耗装饰材料，不降本体 Q。

杯/盘/绘画/钟表：带设施标签（用餐/陈列/时钟/奇观）触发心情/声望逻辑。

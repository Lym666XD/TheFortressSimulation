# 纺织业（古典 / 中世纪 / 文艺复兴）研究与产业链设计（CN）

**版本：** 1.0 　**状态：** 可直接落库实现  
**目标：** 在保持轻量玩法的前提下，把“纤维准备→纺→织→缩呢/漂洗→染整/后整理”拆分为清晰的时代化台阶，并与农业（纤维/染料作物）和畜牧业（羊毛）闭环。

---

## 一、关键史实锚点（用于数值与解锁）
- **主纤维：** 羊毛、亚麻/大麻；（奢侈）丝绸；文艺复兴前后出现**福斯丁**（亚麻经/棉纬）。  
- **设备台阶：** 纺车（中世纪普及）→ 脚踏水平织机（中世纪普及）→ 水力缩呢磨（中世纪）→ 丝车（文艺复兴，水力抛丝/合股）→ 提花织机（文艺复兴，锦缎）。  
- **染整：** 菘蓝（蓝）、茜草（红）、黄草（黄）为常见基础染料；明矾为关键媒染；16 世纪引入**新大陆胭脂虫**（高色力红）。  
- **行业组织：** 行会规程（如佛罗伦萨羊毛行会）带来品质与流程约束。

> 注：本设计不逐条还原史实，仅将上述节点转化为时代解锁与配方差异。

---

## 二、与《矮人要塞》（DF）的差异定位
- **DF 现状：** 植物→纤维线 / 剪毛→毛线 → 织机→布 → 染匠铺→染色 → 裁缝；步骤少、单工坊吞吐。  
- **本设计：** 拆成多段并引入**设备台阶**与**染料/媒染**；通过**吞吐/品质标签**体现价值差异，但仍保持“少表项、好落库”。

---

## 三、时代 → 工艺链 → 投入/产出（IO）→ 解锁

### ① 古典（Classical）
**A. 羊毛呢（手工缩呢）**  
- 链：原毛 → 洗涤/梳理 → 纺纱（纺锤） → 手工织机 → **徒手缩呢** → 晾架整布  
- IO：原毛×10 → 毛纱×8 → 坯呢×6 → 成呢×5  
- 解锁：手工织机（C）、缩呢槽（C）

**B. 亚麻布**  
- 链：亚麻秆 → 沤麻 → 剥麻 → 纺纱 → 手工织机 → 草地漂白  
- IO：麻秆×10 → 纤维×6 → 麻纱×5 → 麻布×5  
- 解锁：麻处理场（C）、手工织机（共用）

**C. 植物染（基础）**  
- 茜草/黄草/菘蓝 + 明矾媒染 → 基础色系。

---

### ② 中世纪（Medieval）
**设备台阶：** 纺车（纱线↑）、**脚踏水平织机**（织造↑）、**水力缩呢磨**（效率↑/质量稳）。

**A. 高档羊毛呢（行会规程）**  
- 链：原毛精选 → 洗/梳 → **纺车** → **脚踏织机** → **水力缩呢** → 拉绒/剪毛 → 定型  
- IO：原毛×10 → 毛纱×9 → 坯呢×7 → 成呢×6（品质+1 标签）  
- 解锁：脚踏织机（M）、水力缩呢磨（M）、剪毛台（M）

**B. 福斯丁（亚麻经/棉纬）**  
- 链：亚麻纱经 + 棉纱纬 → 斜纹/绒面 → 漂白/染色  
- IO：亚麻纱×5 + 棉纱×5 → 福斯丁×8  
- 解锁：福斯丁织造（M）〔棉=贸易输入标签〕

**C. 染整加强**  
- 菘蓝、茜草、黄草标准化栽培；明矾纳入供应。

---

### ③ 文艺复兴（Renaissance）
**设备台阶：** **丝车（filatoio/torcitoio）**、**提花织机（drawloom）**。

**A. 丝绸（抛丝—织造）**  
- 链：蚕茧/生丝 → **丝车抛丝/合股** → 整经 → **提花/缎/天鹅绒织造** → 染整  
- IO：生丝×10 → 丝线×8 → 丝织物×6（奢侈标签/售价↑）  
- 解锁：丝车（R，需水力更佳）、提花织机（R）、丝染房（R）

**B. 胭脂虫红染（高档）**  
- 媒染（明矾）→ 染红 → 皂洗 → 定色（羊毛/丝适用）。

---

## 四、染料与媒染（可直接落库）
- **蓝**：菘蓝（靛还原缸）。  
- **红（旧）**：茜草根 / 地中海胭脂虫（kermes）。  
- **红（新）**：新大陆胭脂虫（cochineal，R 解锁）。  
- **黄**：黄草（weld）。  
- **媒染**：明矾（alum）；基础漂白可用碱液/草灰水。

---

## 五、物品（Items）
- **原料纤维：** 原毛、洁净羊毛、亚麻纤维、麻纤维、生丝/蚕茧、棉纤维（贸易）  
- **半成品：** 毛纱、麻纱、棉纱、丝线、坯布（毛/麻/福斯丁/丝）  
- **染整：** woad_paste、madder_root、weld_yellow、kermes、cochineal、alum、alkali_solution  
- **成品：** 毛呢、亚麻布、福斯丁、丝锦/缎/天鹅绒  
- **副产：** 羊毛脂（→ 肥皂）、缩呢废水（→ 堆肥/事件）

---

## 六、工坊（Workshops）
- **纤维准备：** 洗毛槽（C）、沤麻坑（C）  
- **纺：** 纺锤（C）/ 纺车（M）  
- **织：** 手工织机（C） → 脚踏织机（M） → 提花织机（R，丝）  
- **缩呢/后整理：** 缩呢槽（C） → **水力缩呢磨（M）**、拉绒剪毛台（M）  
- **染整：** 染坊（C/M/R），媒染/漂白台  
- **丝专线：** 丝车（R）、丝染房（R）

---

## 七、代表配方（建议倍率）
- **羊毛线：** 原毛×10 → 洗净×9 → 梳理×8 →（纺锤/纺车）毛纱×8  
- **毛呢：** 毛纱×8 →（脚踏织机）坯呢×7 →（水力缩呢）毛呢×6（+质量）  
- **亚麻布：** 麻秆×10 → 纤维×6 → 麻纱×5 → 亚麻布×5  
- **福斯丁：** 亚麻纱×5 + 棉纱×5 → 福斯丁×8（M 解锁）  
- **丝锦缎：** 生丝×10 → 丝线×8（R 丝车）→（提花）丝织物×6  
- **染色（例）：** 毛呢×6 + alum×1 + woad_paste×1 → 蓝呢×6；毛呢×6 + alum×1 + cochineal×0.5 → 赤呢×6（R 解锁）

---

## 八、玩法与平衡（不做耐久）
- **效率台阶：** 纺车（纱线产能↑）、脚踏织机（织造↑）、水力缩呢（单位劳力产量↑）。  
- **价值台阶：** 丝（丝车/提花）与胭脂虫红赋予“奢侈/礼仪/售价+”。  
- **闭环：** 羊毛脂→肥皂；菘蓝/茜草渣→堆肥；缩呢废水→环保/事件链。  
- **贸易标记：** 棉、胭脂虫、优质明矾可标记为“贸易输入”。

---

## 九、两条专线（科技树摘要）
- **羊毛高端呢（M 核心）：** 洗毛→梳理→**纺车**→**脚踏织机**→**水力缩呢**→拉绒剪毛→染整（菘蓝/茜草/黄草+明矾）。  
- **丝绸（R 奢侈）：** 蚕茧/生丝→**丝车（抛丝/合股）**→整经→**提花织机**→染整（明矾+胭脂虫/茜草）。

---

## 十、与 DF 的最小改造清单
1) 新增建筑：**脚踏织机（M）**、**水力缩呢磨（M）**、**丝车/提花织机（R）**、**扩展染坊配方（明矾、菘蓝缸、胭脂虫）**。  
2) 吞吐：脚踏织机每工单×1.2，水力缩呢每工单×1.3（可调）。  
3) 产地/贸易：棉、胭脂虫、优质明矾标记为“贸易输入”。

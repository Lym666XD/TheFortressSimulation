**Scope:** 反刍/单胃家畜（牛/羊/山羊/猪/马/驴）与家禽（鸡/鸭/鹅）基础生产；饲草—越冬、挤奶—制酪、剪毛、屠宰—皮脂骨副产、蜂箱（可选）。

---

## 1) 核心原则
- **季节性**：夏季产草、秋末屠宰、冬季消耗**干草**维持体况与繁殖率；缺饲草 → 体况/繁殖下降。  
- **饲料优先级**：草地→干草→榨饼/糟渣（次优）→粮食（极端）。  
- **副产闭环**：垫料（秸秆）+粪污 → **堆肥/粪肥**（农业）；**脂肪→油脂/皂化**；**皮→皮革**；**骨→骨针/骨胶**；**蜂蜡/蜂蜜**（可选）。  
- **负担控制**：不做复杂血统与个体遗传，仅设“成熟期/繁殖冷却/胎期/窝产量”的**物种参数**。

---

## 2) 时代 → 工艺链 → 投入/产出 → 解锁

### CLASSICAL（古典）
**A. 基础放牧与畜舍**
- **链路**：牧场放牧 → 饮水槽 → 夜间入舍（简陋）  
- **投入**：青草（地图草量）、水、**秸秆**（垫料，可选）  
- **产出**：**粪肥/堆肥原料**、少量奶（牛/羊/山羊）、鸡蛋、家畜成长  
- **解锁**：**牧场（C）**、**简易畜舍（C）**、**饮水槽（C）**、**小型鸡舍（C）**

**B. 奶与鲜制品**
- **链路**：挤奶 → 简易凝乳（酸/酶） → **鲜奶酪/乳清**（短保）  
- **投入/产出**：生奶×1 → 鲜奶酪×0.6 + 乳清×0.4  
- **解锁**：**挤奶架（C）**、**凝乳盆（C）**

**C. 剪毛与屠宰（基础）**
- **剪毛**：春夏剪毛 → **羊毛**（进入纺织）  
- **屠宰**：成畜 → **肉/皮/脂/骨/内脏**  
- **解锁**：**剪毛架（C）**、**屠宰台（C）**、**盐坛（C，可选）**（轻腌制）

---

### MEDIEVAL（中世纪）
**A. 干草越冬（核心台阶）**
- **链路**：**割草（夏）→ 翻晒 → 干草**（捆/垛） → **冬季饲喂**  
- **投入/产出**：草甸×工期 → **干草**；冬季每成年反刍动物/日消耗 1 单位干草  
- **效果**：有干草 → **体况维持/繁殖正常**；无干草 → **体况下降/繁殖-50%**  
- **解锁**：**草甸/割草工具（M）**、**干草棚（M）**

**B. 畜舍升级与病害控制**
- **链路**：通风/排污沟 → **粪污收集→堆肥坑**  
- **效果**：疾病事件概率下降；**堆肥** +1 肥力（农业挂接）  
- **解锁**：**封闭畜舍（M）**、**堆肥坑（M）**、**粪槽（M）**

**C. 乳制品车间（稳定化）**
- **链路**：挤奶 → 加热/凝乳 → **半硬质奶酪**（保质期↑）  
- **投入/产出**：生奶×1 → 奶酪×0.5 + 乳清×0.5（熟成需时）  
- **解锁**：**奶酪房（M）**、**压酪机（M）**、**盐窖（M，可选）**

**D. 副产加工（效率）**
- **脂肪→油脂/蜡**：熬炼 → **牛羊油/猪油**（烹饪/皂化）  
- **皮→皮革**：浸灰→脱毛→鞣制（与轻工业挂接）  
- **骨→骨针/骨胶**：蒸煮提胶（工艺轻量化）  
- **解锁**：**熬脂锅（M）**、**小型皮鞣槽（M）**、**骨胶锅（M）**

**E. 家禽扩展**
- **链路**：巢箱→产蛋→孵化/食用 → 鸡肥育/鹅肥肝（可选）  
- **解锁**：**鸡舍（M）**、**巢箱（M）**、**孵化器（M/手动）**

---

### RENAISSANCE（文艺复兴）
**A. 饲草优化与副食饲料**
- **链路**：**榨饼/酒糟**→ 配合饲料（少量替代干草）  
- **效果**：冬季**干草缺口≤30%时**，配合饲料可**减轻体况下降**（但成本↑）  
- **解锁**：**饲料拌和台（R）**

**B. 乳制品品质与熟成**
- **链路**：硬质奶酪熟成窖/定湿 → 出品**质量+1 标签**  
- **解锁**：**熟成窖（R）**（只影响售价/士气，数值轻）

**C. 蜂箱（可选轻工业挂接）**
- **链路**：蜂箱 → 取蜜/取蜡 → 蜂蜜/蜂蜡（入酿造/蜡烛/封缄）  
- **解锁**：**蜂箱（R）**、**蜂蜜压榨器（R）**

## 3) 物种参数（示例，落库时可拆表）
- **牛（cattle）**：成熟期长、产奶高、皮厚、冬饲消耗高  
- **羊（sheep）**：产毛、少量奶、耐寒、群居  
- **山羊（goat）**：奶质好、采食性强、对贫瘠地适应  
- **猪（pig）**：不产奶（游戏化忽略），繁殖快、脂率高  
- **马/驴（equids）**：役用为主，可产少量粪肥；（农业侧**重犁/马项圈**已实现）  
- **家禽（chicken/duck/goose）**：蛋/肉；鹅可取绒

（字段建议：`species_id, mature_days, gestation_days, litter_size, daily_feed, winter_hay_need, outputs_tag`）

## 5) 配方（Recipes，摘要）

### 古典（C）
- **挤奶**：`raw_milk ×1`（周期性，需成母畜）  
- **鲜凝乳**：`raw_milk×1 → fresh_cheese×0.6 + whey×0.4`（短保）  
- **剪毛**：`sheep → wool_fleece×1`（季节性）  
- **屠宰**：`adult_livestock → meat×K + raw_hide×1 + tallow/lard×R + bone_bundle×1 + offal×1`

### 中世纪（M）
- **割草—干草**：`meadow_work → hay_bale×N`（受天气影响）  
- **冬季饲喂**：`hay_bale×1/animal/day`（维护体况与繁殖）  
- **半硬质奶酪**：`raw_milk×1 + salt → hard_cheese×0.5 + whey×0.5`（需时）  
- **熬脂**：`fat_raw×1 → tallow/lard×1`  
- **皮鞣（小型）**：`raw_hide×1 + lime/ash + tannin → leather×1`  
- **骨胶**：`bone_bundle×1 → bone_glue×1`  
- **家禽孵化**：`egg×N → chick×p`（几率）

### 文艺复兴（R）
- **配合饲料**：`presscake/spent_grain + hay_bale×0.3 → feed_mix×1`  
- **奶酪熟成**：`hard_cheese → hard_cheese(quality+1)`（时长长）  
- **蜂蜜/蜂蜡**：`beehive harvest → honey×1 + beeswax×1`

## 6) 建筑（Workshops / Areas）
- **牧场（C）**：放牧区域；显示草量  
- **简易畜舍（C）**→**封闭畜舍（M）**：夜间/越冬，疾病几率更低  
- **饮水槽（C）**：供水  
- **挤奶架（C）**、**凝乳盆（C）**：奶与鲜制  
- **剪毛架（C）**：剪毛  
- **屠宰台（C）**：屠宰与分割  
- **草甸（M）**、**干草棚（M）**：割草与储草  
- **堆肥坑/粪槽（M）**：粪污→堆肥  
- **奶酪房/压酪机（M）**：半硬质奶酪  
- **熬脂锅/小鞣槽/骨胶锅（M）**：副产加工  
- **熟成窖（R）**：奶酪熟成（质量标签）  
- **蜂箱/取蜜器（R，可选）**

## 7) 平衡默认值（可直接入表）
- **干草需求**：成年反刍动物 1/日；幼畜 0.5/日；猪/禽 0（使用副食饲料/粮食阈值）  
- **冬季惩罚**：缺饲草 → 体况 −1 级/季、繁殖率 −50%  
- **奶→奶酪**：产率 0.5–0.6；熟成需时长（多季）  
- **剪毛**：每年 1 次；羊毛产量按个体权重（简化）  
- **家禽**：蛋孵化成功率 60–80%（视温度/护理）  
- **病害**：封闭畜舍 & 粪污管理生效时事件−50%  
- **配合饲料**：仅在干草缺口≤30%时发挥作用，效果=减轻体况下降（不提升产量）

## 8) 与其他系统的挂接
- **农业**：`manure/compost → 肥力+`；`hay_bale` 由草甸产出；`straw` 作垫料  
- **纺织**：`wool_fleece → yarn/cloth`  
- **皮革/轻工**：`raw_hide → leather`；`tallow/lard → soap/candle`；`bone_glue → 木工/弓具`  
- **酿造**：`honey → mead`；`spent_grain → feed_mix`（R）

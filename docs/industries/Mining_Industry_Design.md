# 采矿业 — 设计文档（古典 / 中世纪 / 文艺复兴）

> **对应工坊**: [Mine_Site](../workshops/Mine_Site.md)
> 工坊文档是 markdown 实现层 + JSON 对齐；本 industry md 提供历史锚点与设计原理。


**版本**: 1.0  **状态**: 可直接落库实现
**范围**: 露天采矿、地下坑道、矿脉勘探、矿石分类、井下排水/通风/支护、采矿工具时代化。
**Out-of-scope（暂不做）**: 火药爆破（留待化学工坊解锁后再开"晚 R"分支）、深井蒸汽抽水（属于工业革命）、矿权/税收政策。
**Goal**: 把"无中生有的金属锭/原石"补齐为完整的 *勘探 → 开采 → 选矿 → 出矿* 链条，并与冶炼 / 石作 / 化学 / 建筑等下游闭环。

---

## 1) 核心原则

- **时代台阶**: C 期靠人力 + 火裂法 + 简单楔劈；M 期引入水力排水/支护/选矿；R 期引入横坑通风 + 深井卷扬 + 早期化学冶选。
- **三种作业形态**: ①露天/地表 surface ②斜井/平窿 adit/drift ③竖井 shaft。各自有不同的"安全/吞吐/危害"权衡。
- **矿脉系统**: 与现有 MININGSYSTEM_SPEC 衔接；矿脉品位（贫/中/富）影响出矿率，矿种由地质层决定。
- **副产闭环**: 废石 → 骨料（建筑/混凝土）；表土 → 农业回填；选矿尾水 → 事件链（污染/瘟疫钩）。
- **危害与安全**: 塌方 cave-in（已存在物理）、井下水患、缺氧（C/M 用油灯/动物哨兵）、毒气（R 用蜡烛火焰指示）。

## 2) 时代 → 工艺链 → 投入/产出 → 解锁

### 古典（Classical）

**A. 露天采石/采矿（罗马式）**
- 链路: 表土剥离 → 火裂法（柴火加热岩面 + 醋/水骤冷使其碎裂） → 楔劈 → 装篮搬出
- IO: `表土× / 矿层 → raw_stone ×N + ore_* ×N + rubble ×副产`（rubble → 骨料）
- 解锁: **露天矿场（C）**、**火裂作业区（C）**、**人力提篮（C）**

**B. 斜井/平窿（adit）**
- 链路: 顺脉打平窿 → 木支柱（Roman pit prop）→ 油灯照明 → 人力运矿
- IO: 矿脉×工期 → ore_* ×N（产量为露天的 0.7×，但能开采更深的矿）
- 解锁: **平窿入口（C）**、**木支柱（C）**、**油灯/松明（C）**
- 危害: 缺氧（小概率 quarantine 一名矿工 N tick）

**C. 选矿（手选 + 重力分选）**
- 链路: 装锤 → 矿石碎成豆块 → 木盆/木槽水洗 → 手选高品位矿
- IO: `ore_raw ×10 → ore_concentrated ×6 + rubble ×4`
- 解锁: **选矿台（C）**、**洗矿木槽（C）**

---

### 中世纪（Medieval）

**A. 竖井（shaft）+ 卷扬（手摇 → 马拉）**
- 链路: 垂直竖井 → 木质 windlass（手摇绞车）→ 人力或马力卷扬矿筐
- IO: 深层矿脉 → ore_* ×N（深层富矿概率 +15%）
- 解锁: **木竖井 + 井架（M）**、**手摇卷扬（M）**、**马拉卷扬（M，需要畜力）**

**B. 水力排水（核心台阶 — Erzgebirge / Harz 传统）**
- 链路: 矿井水积 → **水力排水轮（M）**抽水（或 Archimedes 螺旋泵）→ 排入溪流
- 效果: 启用排水后，井深上限 +50%；不启用则深井每季有"涌水事件"使矿工死亡/瘫痪
- 解锁: **排水轮（M）**、**Archimedes 螺旋泵（M）**、**排水沟（M）**

**C. 木支护 + 通风（Bergbau 传统）**
- 链路: 标准化木支柱 + 顶板木梁；通风井 + 火炉式自然对流（fire-induced ventilation）
- 效果: 塌方事件 −60%；缺氧事件 −80%
- 解锁: **标准化木支护（M）**、**通风井（M）**、**风炉（M）**

**D. 重力选矿（水力捣碓 + 摇床）**
- 链路: 水力 stamp mill 碎矿 → 摇床 jig 分级
- IO: `ore_raw ×10 → ore_concentrated ×8 + rubble ×2`（比 C 期手选效率高 33%）
- 解锁: **水力捣碓（M）**、**摇床（M）**

**E. 矿工生活区**
- 链路: 矿井口 → 矿工棚 → 集中食宿（缩短上下井时间）
- 效果: 矿工日工时 +25%
- 解锁: **矿工棚（M）**

---

### 文艺复兴（Renaissance — De Re Metallica 时代）

**A. 深井 + 横坑通风（Agricola 1556 范式）**
- 链路: 在原竖井基础上开横向 cross-cut + 通风井；引入大型水轮带动多级抽水链
- 效果: 井深上限再 +50%；可触达深层硫化物 / 银 / 铅 / 锌矿脉
- 解锁: **深井系统（R）**、**多级链泵（R）**、**横坑通风（R）**

**B. 矿测与制图（surveying）**
- 链路: 矿工用 **罗盘 + 链条 + 倾仪** 绘制井下三维图 → 提高出矿效率 / 防误挖
- 效果: 矿脉发现率 +20%；误挖塌方 −30%
- 解锁: **矿测台（R）**、**罗盘 + 链尺（R 装备）**

**C. 化学冶选前奏（与化学工坊挂接）**
- 链路: 黄铁矿/方铅矿/黄铜矿 → 焙烧 → 水洗 → 进入冶炼（不是采矿本身的产出，但是采矿与冶炼的桥梁）
- 解锁: **焙烧炉（R，可与化学工坊共用）**

**D. 早期火药爆破（R 末，可选，需化学工坊解锁黑火药后启用）**
- 链路: 在硬岩处打炮眼 → 装黑火药 → 引线 → 爆破
- 效果: 硬岩开采速度 +60%；触发塌方概率 +20%
- 解锁: **爆破钻孔工位（R，依赖化学工坊黑火药）**
- 注意: 这是封顶配方，符合"R 早期"上限的边界——只在玩家已建造火器工坊 + 化学工坊后开放

## 3) 物品（Items，最小集合）

**原始矿石**: `ore_iron`, `ore_copper`, `ore_tin`, `ore_lead`, `ore_silver`, `ore_gold`, `ore_zinc`, `ore_sulfur`, `ore_saltpeter` (硝石矿)
**金属矿之外**: `raw_stone`（毛石）、`raw_gem`（宝石原石）、`raw_marble`、`raw_granite`、`raw_limestone`、`raw_chalk`、`raw_clay`
**选矿中间品**: `ore_concentrated`（精选矿，进入冶炼）、`rubble`（废石/骨料源）、`mine_tailings`（尾矿，副产）
**辅料**: `timber_prop`（标准化支柱）、`lamp_oil`、`pit_torch`、`miners_tools`（来自金属工坊的采矿工具套件）
**勘探**: `prospect_sample`（勘探样本，给地质学/科学系统）

## 4) 配方（Recipes，摘要）

### 古典（C）
- **火裂开采**: `矿脉 + 柴薪×2 + 水×1 + 劳力 → ore_raw ×N + rubble ×0.5`（@露天矿场，时长 long）
- **平窿开采**: `矿脉（深层）+ 木支柱×2 → ore_raw ×N`（@平窿入口，需 lamp_oil）
- **手选**: `ore_raw ×10 → ore_concentrated ×6 + rubble ×4`（@选矿台）

### 中世纪（M）
- **竖井开采**: `矿脉（深层）+ 木支柱×3 → ore_raw ×N + 深层矿种概率`（@木竖井 + 井架）
- **马力卷扬**: 提矿吞吐 +50%（需马 + 马拉卷扬建筑）
- **水力排水**: 启用 → 井深上限 +50%（@排水轮，需水力）
- **水力选矿**: `ore_raw ×10 → ore_concentrated ×8 + rubble ×2`（@水力捣碓 + 摇床）

### 文艺复兴（R）
- **深井开采**: 启用 → 触达深层稀有矿种（银/锌/硫化物）
- **矿测**: `prospect_sample ×3 + 矿测员劳力 → 区域矿脉图`（增加未探矿脉显示）
- **焙烧**: `ore_sulfide ×5 → ore_roasted ×4 + so2_fume ×副产`（@焙烧炉；硫烟产生 stench/poison 场域）
- **爆破开采（R 末）**: `硬岩 + 黑火药×1 → ore_raw ×N×1.6 + rubble ×N×1.5`（@爆破钻孔工位）

## 5) 建筑（Buildings / Areas）

- **露天矿场（C）**: 区域型，标记矿脉表层
- **平窿入口（C）**、**木支柱（C）**、**油灯/松明（C）**: 浅层地下基础
- **选矿台（C）/ 洗矿木槽（C）**: 手选/水洗
- **木竖井 + 井架（M）/ 手摇 + 马拉卷扬（M）**: 深层地下
- **排水轮（M）/ Archimedes 螺旋泵（M）/ 排水沟（M）**: 水患控制
- **标准化木支护（M）/ 通风井（M）/ 风炉（M）**: 安全
- **水力捣碓（M）/ 摇床（M）**: 选矿
- **矿工棚（M）**: 生活
- **深井系统（R）/ 多级链泵（R）/ 横坑通风（R）**: 深层
- **矿测台（R）**: 勘探
- **焙烧炉（R）**: 化学冶选前置
- **爆破钻孔工位（R，可选）**: 火药开采

## 6) 平衡默认值（可直接入表）

- **基础出矿率**: 露天 ×1.0；平窿 ×0.7；竖井 ×1.2（深层）；深井 ×1.4（稀有矿）
- **选矿回收率**: 手选 60%；水力 80%；化学焙烧（R）90%
- **排水启用**: 井深上限 ×1.5；未启用 → 每季 5% 涌水事件
- **塌方基线**: 5% / 季 / 矿井；M 支护 −60%；R 矿测 −30%
- **缺氧 / 毒气**: C/M 概率 3% / 矿工 / 季；通风井 −80%
- **马拉卷扬**: 吞吐 ×1.5（vs 手摇）；占用 1 头马
- **爆破（R 末）**: 速度 +60%；塌方 +20%；需化学工坊解锁黑火药

## 7) 与其他系统的挂接

- **冶炼工业**: `ore_concentrated` 是冶炼的唯一合法输入（贫矿可入但低产）
- **石作**: `raw_stone / raw_marble / raw_granite` 直接进入 stonecutter
- **建筑**: `rubble` 进入混凝土场作 aggregate；`raw_clay` 进入砖瓦窑
- **化学/炼金**: `ore_sulfur` / `ore_saltpeter` → 火药与酸；`ore_alum` → 媒染
- **林业**: 消耗大量 `timber_prop` 与 `charcoal_std`（火裂法）
- **畜牧**: 马拉卷扬消耗马
- **MININGSYSTEM_SPEC**: 本文档定义"如何采"，MININGSYSTEM 定义"地下结构 + 矿脉数据 + 玩家挖掘指令调度"，两者必须对齐

---

## 8) 与 DF 的差异定位

- DF 一刀切的"挖一格 → 出一堆石头"被替换为 **矿脉品位 + 选矿率 + 排水/支护/通风约束** 的轻量系统
- 矿测系统让"探矿"成为玩法（DF 几乎没有勘探乐趣）
- 副产闭环 `rubble → aggregate / mine_tailings → 事件` 增加现实感

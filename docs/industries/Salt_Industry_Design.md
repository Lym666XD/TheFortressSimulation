# 采盐业 — 设计文档（古典 / 中世纪 / 文艺复兴）

> **对应工坊**: [Salt_Works](../workshops/Salt_Works.md)
> 工坊文档是 markdown 实现层 + JSON 对齐；本 industry md 提供历史锚点与设计原理。


**版本**: 1.0  **状态**: 可直接落库实现
**范围**: 海盐（晒板 / 煮盐）、岩盐矿、内陆盐井 / 卤泉、graduation tower（晚 M / R）、盐贸易。
**Out-of-scope**: 现代离子膜电解、真空制盐（工业革命后）。
**Goal**: 把"烹饪 / 皮革 / 畜牧 / 渔业 / 化学全都用盐却没人产"的空白补上；体现地理决定性（沿海 vs 内陆 vs 山区岩盐区）；与贸易/事件挂接（盐税、盐道、盐之乱）。

## 1) 历史锚点

- **C 罗马**: 地中海盐田（salina）— 引海水进入多级浅塘，太阳蒸发 → 结晶塘收盐；Via Salaria（盐道）从台伯河口运盐到罗马；Hallstatt 奥地利岩盐矿自 BC 1500 起持续作业（"盐"hall 一词在凯尔特/日耳曼地名中无数次出现）。
- **M 中世纪**: Wieliczka 波兰岩盐矿（13c 起，世界最古老连续作业岩盐矿之一）；德意志 Lüneburg、Halle 城市靠卤水煮盐（Sieden）兴起；威尼斯靠盐业垄断成富商；法国 / 葡萄牙大西洋沿岸 multi-pond salt works。
- **R 早期**: Salzkammergut（奥地利 / 巴伐利亚）大规模国家垄断；**Graduation tower（Gradierwerk）**— 用黑刺李枝堆成的塔，让低浓度卤水反复淋下，借风蒸发浓缩——可大幅减少燃料消耗；铁锅煮盐技术成熟。

## 2) 时代 → 工艺链 → 投入/产出 → 解锁

### 古典（C）

**A. 地中海盐田（solar salt works）— 沿海地图核心**
- 链路: 引海水 → 第一级蒸发塘（浅）→ 第二级浓缩塘 → 结晶塘 → 人工耙盐 → 装篮
- IO: `seawater（地图无限）+ 烈日工期 → salt_coarse ×N`（依赖季节 + 天气）
- 解锁: **盐田（C，仅可建在沿海/咸水湖边）**
- 季节性: 夏季产量 ×3，秋冬 ×0.3（雨季几乎停产）

**B. 岩盐矿（rock salt mining）— 山区**
- 链路: 在岩盐层中开采（与采矿系统挂接）→ 大块岩盐 → 破碎 → 装袋
- IO: 岩盐矿脉 → `rock_salt ×N + 副产 rubble ×小`
- 解锁: 与采矿工坊联动；**岩盐场（C）** = 露天矿场 + 盐工
- 优势: 不受季节影响；高产；地理稀有

**C. 煮盐（sea / brine, simple）— 北方沿海 / 内陆卤泉**
- 链路: 取卤水（海水 / 盐泉）→ 陶罐 / 铅锅煮干 → 盐
- IO: `brine ×10 + charcoal ×3 → salt_coarse ×2`
- 解锁: **煮盐灶（C，需柴薪）**
- 注: 燃料密集；C 期效率低，作内陆地图替代方案

---

### 中世纪（M）

**A. 多级盐田（Atlantic / 法国大型版）**
- 链路: 地中海 C 版的工业化放大；筑堤、闸门、多级回灌
- IO: 比 C 期吞吐 +50%
- 解锁: **大型盐田（M，沿海地形需求强）**、**盐田闸门（M）**

**B. 卤水井 + 煮盐城（Sieden — Lüneburg / Halle 范式）**
- 链路: 钻井 → 卤水 → 木 / 铅盐锅大火煮 → 撇盐 → 装袋
- IO: `brine ×10 + charcoal ×2.5 → salt_coarse ×2.5`（比 C 煮盐效率 +25%）
- 解锁: **卤水井（M）**、**盐锅煮房（M，专用建筑）**
- 注: 城市经济引擎；与建筑业"专用盐城"主题挂接（可作"盐之城"特色）

**C. 岩盐深井（Wieliczka 范式）**
- 链路: 深竖井 + 卷扬 + 排水（依赖采矿 M 期排水轮）→ 深层岩盐
- IO: 深层岩盐矿脉 → `rock_salt ×N×1.5`（深层富岩盐 +50%）
- 解锁: **岩盐深井（M，依赖采矿 M）**

**D. 盐路 + 盐税（事件 / 贸易）**
- 不是产业链节点，是**贸易与事件**：
  - 盐道商队 → 把盐运到内陆获取高价
  - 盐税官 → 周期性事件（征税或抗税）
  - 盐之乱 → 盐价波动事件（Director 触发）
- 解锁: **盐仓（M）**、**盐商队驿（M）**（与贸易系统挂接）

---

### 文艺复兴（R）

**A. Graduation Tower（Gradierwerk）— R 期标志**
- 链路: 木架塔填满黑刺李 / 树枝 → 低浓度卤水泵到塔顶 → 沿树枝缓慢淋下 → 风 + 阳光蒸发浓缩 → 收集到底槽 → 减少后续煮盐燃料 80%
- IO: `brine_weak ×10 → brine_strong ×3 + 时长 long`；后续煮盐燃料 ×0.2
- 解锁: **Gradierwerk 浓卤塔（R，需开放风口地形）**
- 影响: 卤水煮盐效率提升 — 比 M 期再 +60% 总产量；可建多座并联

**B. 铁锅煮盐（Iron pan boiling）**
- 链路: 用大型铸铁盐锅代替木 / 铅锅 → 更高温 → 更快
- IO: `brine ×10 + charcoal ×1.5 → salt_fine ×3`
- 解锁: **铁锅煮房（R，需冶炼 M 期生铁锅）**
- 产物: salt_fine（比 coarse 高一档：质量 +1，烹饪 / 保藏 +10% 效果）

**C. 盐精化（salt refining）**
- 链路: 粗盐 + 水溶解 → 沉淀杂质 → 再蒸发 → 食盐 / 药用盐
- IO: `salt_coarse ×3 → salt_fine ×2 + brine_residue ×1`
- 解锁: **精盐工台（R）**

**D. 香料盐 / 配方盐（与烹饪 R 期挂接）**
- 已经在烹饪 R 期 Spice & Sugar Bench 出现；本工坊提供原料 salt_fine。

## 3) 物品（Items）

- **原料**: `seawater`（地图属性）、`brine_weak / brine / brine_strong`、`rock_salt_raw`
- **半成品**: `salt_coarse`（粗盐）、`salt_fine`（精盐）
- **副产**: `brine_residue`（精盐副产 → 化学杂用）、`bittern`（卤水母液，含镁 — 长线化学输入）

## 6) 平衡默认值

- **地理决定**: 盐田仅可建于"沿海"或"咸水湖"地形；岩盐场需要"岩盐矿脉"地质 tag；卤水井需要"盐泉"地图属性
- **季节性**: 盐田夏 ×3 / 秋冬 ×0.3；岩盐场恒定
- **燃料消耗**: 煮盐 C → M → R 每单位盐燃料 = 1.5 / 1.0 / 0.3（Gradierwerk 加成）
- **salt_fine vs coarse**: 烹饪保藏效果 ×1.1；售价 ×1.5
- **贸易**: 内陆地图盐价 ×3（默认）；可触发"盐道商队"事件

---

## 7) 与其他系统的挂接

- **烹饪**: 所有腌制 / 调味 / 鱼酱 / 香料盐都消耗 salt
- **皮革**: 白革（alum + salt）需要盐
- **畜牧**: 奶酪 / 腌肉
- **渔业**: 腌鱼 / 熏鱼
- **化学**: bittern → 杂物盐化学；与硝石田 (manure + ash + lime + 时长) 间接相关
- **贸易**: 盐 = 最常见的内陆刚需贸易品
- **冶炼**: 提供生铁锅（R 铁锅煮房）
- **林业**: 燃料；木架 graduation tower 需要木材
- **建筑**: 盐城作为城镇主题（Halle / Lüneburg / Hallstatt 风）

---

## 8) 与 DF 的差异

- DF 盐基本不可见、不可产；本设计补齐"地理 → 产能 → 贸易"完整循环
- 引入 Gradierwerk 作 R 期标志台阶；引入"盐城"主题给建筑业故事感
- 不复制现代离子工艺，保持时代纯度

# 建筑业（陶瓷 / 砖瓦 / 石灰 / 屋面）简化设计 — 脱胎于 DF，按你五条要求重构

> **对应工坊**: [Stoneworks](../workshops/Stoneworks.md) · [Pottery](../workshops/Pottery.md)
> 工坊文档是 markdown 实现层 + JSON 对齐；本 industry md 提供历史锚点与设计原理。


**版本：** 1.0 　**状态：** 可直接落库实现  
**核心变更：** 原矿石/毛石**不能直接建墙**；必须在工坊加工为 **stone block** 或使用产业链建材（brick / mortar / roof tile / concrete mix 等）。**不实现耐久**数值，改用建造速度、搬运效率、美观/舒适与功能标签体现优势。

---

## 一、变更原则（对 DF 的“断舍离”）
1. **禁止**：开采出的原矿石、毛石直接“建墙”。  
2. **强制**：石材须在**石材工坊（Stonecutter）**加工成 **stone block**；或使用产业链建材（砖、灰浆、混凝土、瓦）。  
3. **简化**：每个时代仅保留 **1–2 条**代表性建造方式，统一物品与配方。  
4. **沿用 DF**：暂不设计耐久破坏。  
5. **优势导向**：新建材在**建造速度 / 搬运效率 / 美观 / 舒适 / 功能**上显著优于“毛石直砌”。

---

## 二、时代 → 建造套件（Build Set）→ IO → 解锁

> 说明：每条 Build Set 是一个**可选施工模式**；不强制，但选择即获得相应加成（见第六节）。配方数量为**默认建议**，便于直接落表。

### A. 古典（Classical）

**Set C1：石灰砌块墙（Air‑Lime Masonry）**  
- **IO**：`stone_block ×10 + mortar_air ×1 → 墙面（10段）`  
- **解锁**：**Lime Kiln（石灰窑, C）**、**Slaking Pit（熟化坑, C）**、**Mortar Trough（灰浆槽, C）**  
- **说明**：基础的“块 + 空气硬化灰浆”砌筑。

**Set C2：罗马砖与屋面（Brick & Tile）**  
- **墙 IO**：`brick ×20 + mortar_air ×1 → 墙面（10段）`  
- **屋面 IO**：`roof_tile ×25 + mud_or_mortar ×1 → 屋面（25格）`  
- **解锁**：**Clamp Kiln（夹窑, C，区域）** 或 **Updraft Kiln（上吸窑, C）**，**Roofing Yard（屋面作业区, C）**

**Set C3（资源可选）：水硬灰浆 / 罗马混凝土**  
- **灰浆 IO**：`slaked_lime ×1 + pozzolan ×1 + sand ×2 → mortar_hydraulic ×6`  
- **地坪/道路 IO**：`concrete_mix ×10 → 混凝土地坪（10格）`（`concrete_mix = slaked_lime + pozzolan + aggregate`）  
- **解锁**：**火山灰 pozzolan** 或 **crushed_brick（碎砖粉, C）**，**Mixing Yard（拌合场, C）**  
- **定位**：用于**排水/浴场/水工设施**与**地坪/路面**。

---

### B. 中世纪（Medieval）

**Set M1：批量砖石（Batch Brickwork）**  
- **墙 IO**：`brick ×24 + mortar_air ×1 → 墙面（12段）`（相较 C2 **多 2 段**，体现模板与批量施工）  
- **解锁**：**Updraft Kiln（上吸窑, M—效率↑）**；**Mortar Trough**（共用）

**Set M2：木构‑抹泥填充（Timber Frame Infill）**  
- **墙 IO**：`timber_frame ×1 + daub_mix ×2 → 墙面（6段）`  
  其中 `daub_mix = clay + straw + slaked_lime（少量）`  
- **解锁**：**Carpentry Frame（木工架, M）**、**Daub Bench（抹泥台, M）**  
- **定位**：**最快**、**最省砖石/石灰**，但**美观低**（适合内墙/乡野建筑）。

**Set M3：北式屋面（Peg Tile Roof）**  
- **IO**：`roof_tile ×25 + wood_pegs ×25 → 屋面（25格）`  
- **解锁**：**Peg Bench（木钉作坊, M）**、**Roofing Yard（共用）**

---

### C. 文艺复兴（Renaissance）

**Set R1：精整砌筑（Refined Masonry）**  
- **墙 IO**：`stone_block ×10 + mortar_hydraulic ×1 → 墙面（10段，质量+1）`  
- **解锁**：**Updraft Kiln（R）**（稳定出砖瓦）、**改良灰浆配方（同 C3）**

**Set R2：赤陶饰面 / 彩绘地砖（Terracotta & Decorative Tiles）**  
- **立面 IO**：`terracotta_piece ×4 + mortar_air ×1 → 饰面（4单元）`  
- **地面 IO**：`decor_tile ×16 + glaze ×1 → 地面（16格）`  
- **解锁**：**Terracotta Bench（赤陶作坊, R）**、**Glaze Bench（施釉台, R）**

## 三、统一物品（最小集合）

- **块材/砖瓦**：`stone_block`、`brick`、`roof_tile`、`decor_tile`、`terracotta_piece`  
- **灰浆/混凝土**：`quicklime`、`slaked_lime`、`mortar_air`、`mortar_hydraulic`、`concrete_mix`  
- **填充/木构**：`timber_frame`、`daub_mix`（= clay + straw + slaked_lime）  
- **辅料**：`sand`、`aggregate`、`pozzolan（或 crushed_brick）`、`glaze`、`wood_pegs`、`mud_or_mortar`

**运输与堆叠（优势设置）**  
- `stone_block`：**10/堆**；`brick`/`roof_tile`：**25/堆**；`mortar_*`：**1 桶≈10 段墙**。显著减少搬运次数＝**总体工期缩短**。

## 四、配方（简化可落表）

**石材加工（全时代）**  
- `raw_stone ×1 → stone_block ×2 @Stonecutter`

**石灰链（C→R）**  
- `limestone/chalk ×1 + fuel ×1 → quicklime ×1 @Lime Kiln`  
- `quicklime ×1 + water ×1 → slaked_lime ×1（需时） @Slaking Pit`  
- `slaked_lime ×1 + sand ×2 → mortar_air ×6 @Mortar Trough`

**水硬灰浆 / 预拌混凝土（C/R）**  
- `slaked_lime ×1 + pozzolan ×1 + sand ×2 → mortar_hydraulic ×6 @Mortar Trough`  
- `slaked_lime ×1 + pozzolan ×1 + aggregate ×3 → concrete_mix ×6 @Mixing Yard`

**砖与瓦（C→R）**  
- `clay ×1 + fuel ×1 → brick ×2 @Clamp/Updraft Kiln`  
- `clay ×1 + fuel ×1 → roof_tile ×2 @Clamp/Updraft Kiln`  
- `fine_clay ×1 + glaze ×1 + fuel ×1 → decor_tile ×2 @Updraft Kiln`

**木构/抹泥（M）**  
- `timber ×2 → timber_frame ×1 @Carpentry`  
- `clay ×1 + straw ×1 + slaked_lime ×0.5 → daub_mix ×2 @Daub Bench`

## 五、建筑与解锁（极简）

- **Stonecutter（全时代）**：毛石 → stone_block  
- **Lime Kiln（C）** → **Slaking Pit（C）** → **Mortar Trough（C）**  
- **Clamp Kiln（C，区域） / Updraft Kiln（M→R）**：brick、roof_tile、decor_tile  
- **Roofing Yard（C→R）**：铺瓦施工  
- **Carpentry & Daub Bench（M）**：木构‑抹泥填充  
- **Mixing Yard（C/R）**：concrete_mix（可与 Mortar Trough 合并为“拌合场”）  
- **Terracotta/Glaze Bench（R）**：立面饰件与彩绘地砖

## 六、建造模式与加成（不涉及耐久）

| 类型 | 施工对象 | 时代 | 主要输入 | **建造速度** | **搬运负担** | **美观/舒适** | **功能标签** |
|---|---|---|---|---|---|---|---|
| **块石（干式）** | 墙 | 全时代 | stone_block | **+15%**（标准化切块） | ↓（10/堆） | +1 | — |
| **块石 + 空气灰浆** | 墙 | C→R | stone_block + mortar_air | +10% | ↓ | +2（勾缝整齐） | — |
| **砖墙（批量模板）** | 墙 | M | brick + mortar_air | **+20%** | **↓↓**（25/堆） | +2 | — |
| **木构‑抹泥** | 墙 | M | timber_frame + daub_mix | **+30%**（最快） | ↓ | 0（乡野风）/ 室内舒适+1 | — |
| **水硬砌筑** | 墙 | C/R | stone/brick + mortar_hydraulic | +10% | ↓ | +2 | **可建水工/潮湿区** |
| **混凝土地坪/道路** | 地面 | C/R | concrete_mix | **+30%**（浇筑） | ↓ | +1 | **防泥/卫生+** |
| **瓦屋面（通用）** | 屋面 | C→R | roof_tile (+ pegs/M) | +15% | **↓↓** | +2 / 室内舒适+1 | **防雨/静音+** |
| **赤陶饰面/彩砖** | 立面/地面 | R | terracotta / decor_tile | +10% | ↓ | **+3（豪华）** | — |

> **说明**：将“石块无灰浆砌筑”视为 0% 基线；以上加成均以此为相对值。

## 七、与“传统 DF 方式”的对比（新体系的优势）
- **必须先加工**（石材→stone_block）：换来 **+15% 以上的建造速度**、**美观+1**、**更少搬运**（10/堆），流程更清晰。  
- **砖与瓦的批量化**：**堆叠大**（25/堆）、批量模板减少工单；屋面提供 **舒适+1 / 观感+2**。  
- **灰浆是“开关式收益”**：空气灰=勾缝即美观；水硬灰=**可在潮湿/水工区域施工**（喷泉、浴室、排水沟、河岸）。  
- **混凝土地坪/道路**：**+30% 施工速度**、不积泥、厨卫/作坊“卫生+”，显著优化流线与清洁。  
- **R 期装饰**：赤陶与彩砖提供 **+3 豪华度** 与“城市景观”，不涉及耐久与复杂结构。

---

## 八、实现建议（最小代价）
1. **移除**“原矿石/毛石直接建墙”工作，所有墙体任务均指向 **stone_block / brick** 等材料。  
2. **加入** 9 大类核心物品：`stone_block / brick / roof_tile / mortar_air / mortar_hydraulic / concrete_mix / daub_mix / terracotta_piece / decor_tile`。  
3. **吞吐单位硬规则**：10 段墙 ≈ 1 桶灰；25 砖/瓦 = 1 堆 —— 显著减少工单和搬运。  
4. **功能标签优先**：`waterworks / hygiene / comfort / noise` 等先落地，**耐久留待后续版本**。  
5. **UI 友好**：建筑菜单里将施工方式分为 **块石 / 砖 / 木构 / 混凝土 / 屋面 / 饰面** 六个按钮，每个按钮下展示加成提示与所需物品。

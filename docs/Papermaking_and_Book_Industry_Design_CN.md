## 0) 扼要结论（TL;DR）
- **书写载体更替**：地中海以**纸草（papyrus）**与**羊皮纸/犊皮纸（parchment/vellum）**为主，罗马晚期到中世纪形成**卷轴→抄本（codex）**的过渡；纸张由**中国→伊斯兰世界（8–9 世纪）→欧洲（12–13 世纪）**传播，**法布里亚诺（Fabriano）**成为欧洲早期纸业中心并引入**水印/动物胶施胶/水力捣槌**等关键创新。  
- **书籍生产方式**：古典—中世纪依赖**抄写室（scriptorium）**手写、上色、装订；文艺复兴中叶后，**谷登堡活字印刷**以**铅–锡–锑字模、油性印刷油墨、螺旋/杠杆压印机**实现成本骤降与规模化；**伦敦文书业公会（Stationers’ Company, 1557）**建立出版监管与行业规制。  
- **与 DF 的差异**：DF 具备**纸/羊皮纸→卷册（quire）/书卷（scroll）→抄本（codex）**基础链，但**没有印刷**，纸张由“植物纤维→纸浆→螺旋压机”简化而来，书写依赖**学者在图书馆**自发生成文本。我们在不大幅增负担的前提下，引入**轻量的造纸车间分段**与**印刷所（文艺复兴）**，并把**水印/施胶/装订结构**做成品质与产能的“台阶”。

## 1) 历史脉络（用于解锁与配方锚点）
### 古典（Classical）
- **载体**：埃及及地中海常用**纸草**；同时发展出**羊皮纸/犊皮纸**（石灰浸渍→拉伸刮削→干燥整饰）。  
- **书形**：从**卷轴**向**抄本（codex）**过渡，1–2 世纪已有早期抄本实例；基督教与学术需求助推抄本普及。  
- **书写与颜料**：**铁胆墨（oak gall + 铁盐 + 阿拉伯胶）**逐步成为西方最重要书写墨（早期亦有碳墨），金箔/彩绘用于装饰。  

### 中世纪（Medieval）
- **手工造纸传入欧洲**：纸张起源于中国，8 世纪传至**撒马尔罕**与巴格达，12–13 世纪经**西西里/西班牙/意大利**入欧；欧洲采用**麻/亚麻旧布**为纤维，经**水力捣槌**制浆，使用**金属丝布浆网（laid mould）**成形，**动物胶施胶**提高强度与书写性，并**压印水印**识别纸坊。**法布里亚诺**在 13 世纪形成系统工艺并辐射全欧。  
- **书籍制作**：大学兴起后，书籍需求增长；装订以**缝在皮条/麻绳上→穿板（木板封面）→皮革包面**为常态，常见**金属扣/包角**。  
- **书写墨**：铁胆墨成为书写标准。  

### 文艺复兴（Renaissance）
- **印刷革命**：约 1450s **谷登堡**整合**耐用字模合金（Pb–Sn–Sb）、油性墨、机械压印机**；**活字 + 纸**使**单本成本**与**复制时间**大幅下降，推动纸业扩张。  
- **行业规制**：1557 年**Stationers’ Company**获皇家特许在伦敦垄断印书与登记权，形成早期“书商版权/版面登记”。  
- **装订**：向**纸板（pasteboard）**与更轻便的结构演化，但仍常见“缝在槽锯绳（sawn-in）或外置麻绳（sewing on cords）”两类。  

> 参考节点：羊皮纸工艺；纸张传播与欧洲工艺；法布里亚诺创新（水印/施胶/捣槌）；铁胆墨；谷登堡三件套；Stationers’ Company。

## 2) 《矮人要塞》对比（现状抓要点）

## 3) 你的游戏化设计（轻量升级，沿用“时代→工艺链→投入/产出→解锁”）

### 3.1 古典（C）
**A. 羊皮纸（Parchmenter’s Frame）**  
- **链**：生皮 →（灰浆浸泡）→ 拉伸刮削 → 晾干抛光 → **羊皮纸张**  
- **IO**：生皮×1 + 灰浆×1 → 羊皮纸×2（品质随工匠/抛光度）  
- **解锁建筑**：**皮纸架（C）**、**抄写台/小抄写室（C）**  
- **衔接**：副产**胶料/边角**→ 胶水/明胶；入**装订/绘制**。  

**B. 抄写抄本（Scriptorium）**  
- **链**：羊皮纸/纸草 → 抄写 → 成册（缝在皮条）→ 简装抄本  
- **IO**：羊皮纸×5 + 亚麻线×1 + 薄木板×2 → 抄本（手写）×1（价值=材质+工艺）  
- **解锁**：**抄写室（C）**、**简装装订台（C）**  

> **玩法**：C 期以**耐久/尊贵**为卖点（羊皮纸），成本高、产能低。

---

### 3.2 中世纪（M）
**A. 手工纸（Rag Paper, Laid）**  
- **链**：破布分拣 → 水力捣槌**制浆** → 抄纸（**金属丝网模+框坎**）→ 覆毡压榨 → 干燥阁 → **动物胶施胶** → 打磨  
- **IO**：旧布×10 + 清水×N → 纸张（Laid）×12（可带**水印**标签）；施胶消耗**明胶×0.1**/张  
- **解锁建筑**：**纸坊（水力）M**、**干燥阁（M）**、**施胶槽（M）**  
- **品质**：有施胶/水印 → **书写性+1 / 售价+1**；无施胶 → 易渗墨。  

**B. 装订升级（Sewing on Cords）**  
- **链**：纸张/羊皮纸 → 折帖成册 → 缝在麻绳/皮条 → 穿板（木板封面）→ 皮革包面/金属扣  
- **IO**：册帖×8 + 麻绳×2 + 木板×2 + 皮革×1 → 手工精装抄本×1（**耐久+**）  
- **解锁**：**装订作坊（M）**（含缝制治具/压槽）  

**C. 抄写/彩绘**（可选轻量）  
- **链**：铁胆墨/颜料 → 抄写/题红 → 简易镀金  
- **IO**：墨水×0.1 + 颜料×0.1 → 装饰页面（售价↑）  
- **来源**：**铁胆墨**=**橡瘿单宁 + 绿矾 + 阿拉伯胶**；颜料来自**染整/化学**。

> **玩法**：M 期可批量造**高品质纸张**，装订更坚固；图书更易流通。

---

### 3.3 文艺复兴（R）
**A. 活字印刷（Printing House）**  
- **链**：**字模合金（Pb–Sn–Sb）**浇铸 → 组版 → **油性墨**上墨 → 压印 → 晾干 → 折帖 → **装订**  
- **IO**：字模套装×1 + 纸张×20 + 油墨×1 → **印张×20** → 折帖装订 → **印刷书×2–3 本**  
- **解锁建筑**：**铅字铸字台（R）**、**印刷所（R）**（含压印机/上墨台）、**装订所（升级）**  
- **品质/效率**：与手抄相比，**产能×10+**；需要**油墨/字模维护**产线保障。  

**B. 行业规制（轻量规则）**  
- **机制**：建造**“书籍登记台（Stationer’s Desk）”**后，批量印刷需“登记”（消耗纸/费用极少，换取**交易溢价/扣押风险↓**）。  
- **来源**：仿**Stationers’ Company (1557)** 的登记与监管传统。

> **玩法**：R 期开启**规模化知识生产**与**贸易**；引入**字模/油墨/登记**的低负担管理。

## 5) 与既有系统的挂接
- **化学**：绿矾/铁胆墨；动物胶（与皮革副产）；油性油墨需**亚麻籽油/松香**（你已有油脂/化学线可复用）。  
- **纺织**：旧布/破布来源（福斯丁/亚麻/棉）；麻绳/线。  
- **冶金**：字模合金（Pb–Sn–Sb）；印刷机金属件；纸坊水轮与建筑木构件。  
- **畜牧**：生皮→羊皮纸；骨胶/明胶。

## 6) 与 DF 的落地差异（最小改造）
- **新增质量环节**：手工纸需**施胶/干燥阁**；是否带**水印**决定售价/鉴伪。  
- **新增产能台阶**：纸坊（水力）、印刷所（R）提供**明显吞吐倍率**；手抄与印刷并存。  
- **轻量监管**：登记台 = 贸易与治安小增益，不引入复杂法律系统。

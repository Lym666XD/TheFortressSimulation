## 1）与 DF 的差异定位（快速）
- **DF 现状：** 一个 **Kitchen** 负责“Prepared Meal（简/中/豪华）”与**炼脂成牛羊油/猪油**；料理提升价值与心情。  
- **本设计：** 拆为 **灶锅（煮/炖）**、**烤炉（烤/烘）**、**腌熏屋（保藏）** 三件套，文艺复兴期新增**香料与糖台**；每条仅保留 1–2 个代表配方。

---

## 2）总体结构
- **四条通用工艺**  
  ① **煮/炖**（Cauldron）：日常口粮（puls/pottage/stew）。  
  ② **烤/烘**（Oven/Hearth）：面包/烤饼/（R期）馅饼。  
  ③ **保藏**（Saltery/Smokehouse）：盐腌/熏制/醋渍（远征与越冬核心）。  
  ④ **调味/调脂**（Condiments & Fats）：鱼酱/香料盐/甜香糖浆；橄榄油（地中海）、猪油/黄油（北方）。

- **三座（+一）工坊**  
  - **Hearth & Cauldron（C）**：煮/炖。  
  - **Hearth Oven（C） → Communal Oven（M）**：烤饼/面包/（R）馅饼。  
  - **Saltery / Smokehouse（M）**：腌制/熏制/醋渍。  
  - **Spice & Sugar Bench（R）**：香料与糖浆（风味/宴会加成）。

- **数值表达（建议）**  
  - **煮/炖**：份数↑、“热食/饱足”心情+1。  
  - **烘焙**：携带性/合餐效率↑（工作队吃饭节拍更顺）。  
  - **腌/熏/醋渍**：保质↑↑（远征/冬季保障）。  
  - **调味/油脂**：风味+1，部分配方触发“宴会/筵席”标签。

## 3）时代 → 工艺链 → 投入/产出（IO）→ 解锁

### A. 古典（Classical）
**C1｜一锅炖（Pottage / Puls）**  
- **链**：谷物/豆类 + 蔬菜 + 水 → 文火熬煮。  
- **IO**：谷×4 + 豆/菜×2 + 水×1 → **热炖 ×8 份**（与 DF 一致：烹饪不返种子）。  
- **解锁**：**Hearth & Cauldron（C）**。

**C2｜面饼/扁面包**  
- **链**：磨粉 → 和面 → 烤炉/热石烘烤。  
- **IO**：面粉×2 + 水×1 + 盐×0.1 → **扁面包 ×4**。  
- **解锁**：**Hearth Oven（C）**。

**C3｜鱼酱（调味）**  
- **链**：小鱼/内脏 + 粗盐（曝晒发酵）→ 滤取酱汁。  
- **IO**：鱼料×4 + 粗盐×2 → **鱼酱 ×1**（作为“调味品”消耗：任一炖/烤成品 **风味+1**）。  
- **解锁**：**Fermentation Jar（C）**。

---

### B. 中世纪（Medieval）
**M1｜公共面包炉（批量）**  
- **链**：面团集中入炉大批量烘焙（节柴/吞吐↑）。  
- **IO**：面团×6 + 柴薪×1 → **大面包 ×8**（吞吐优于 C 期）。  
- **解锁**：**Communal Oven（M）**。

**M2｜腌/熏/醋渍（保藏）**  
- **腌肉**：肉×4 + 盐×1 → **腌肉 ×4（保质↑↑）** @Saltery。  
- **熏鱼**：鱼×4 + 木屑×1 → **熏鱼 ×4（保质↑↑）** @Smokehouse。  
- **醋渍菜**：蔬×3 + 醋×1 + 盐×0.5 → **醋渍蔬菜 ×4（保质↑）** @Saltery。  
- **解锁**：**Saltery / Smokehouse（M）**。

**M3｜浓炖（加脂）**  
- **链**：与 C1 相同，加入少量**动物脂肪**（猪油/黄油）。  
- **IO**：谷×4 + 菜×2 + 脂×0.5 + 水×1 → **浓炖 ×8（风味+1）**。  
- **解锁**：共用 **Hearth & Cauldron**。

---

### C. 文艺复兴（Renaissance）
**R1｜香料与糖（宴席/风味）**  
- **链**：糖 + 香料（肉桂/肉豆蔻/胡椒混合）→ 甜香糖浆 / 香料盐。  
- **IO**：糖×1 + 香料×0.5 → **甜香糖浆 ×1** *或* **香料盐 ×1**（为菜肴添加 **风味+1 / 宴会+1**）。  
- **解锁**：**Spice & Sugar Bench（R）**。

**R2｜馅饼（宴席类烘焙）**  
- **链**：肉/蔬 + 面团（包馅）→ 烘烤。  
- **IO**：馅×3 + 面团×2 → **馅饼 ×5（宴会+1）** @Bake Oven（R）。

## 6）代表配方（倍率建议）
- **炖 C**：谷×4 + 菜×2 + 水×1 → **热炖 ×8**（@Hearth & Cauldron）。  
- **炖 M**：谷×4 + 菜×2 + 脂×0.5 + 水×1 → **浓炖 ×8（风味+1）**（@Hearth & Cauldron）。  
- **扁面包 C**：面粉×2 + 水×1 + 盐×0.1 → **扁面包 ×4**（@Hearth Oven）。  
- **大面包 M**：面团×6 + 柴薪×1 → **面包 ×8**（@Communal Oven，批量+）。  
- **馅饼 R**：馅×3 + 面团×2 → **馅饼 ×5（宴会+1）**（@Bake Oven）。  
- **腌肉 M**：肉×4 + 盐×1 → **腌肉 ×4（保质↑↑）**（@Saltery）。  
- **熏鱼 M**：鱼×4 + 木屑×1 → **熏鱼 ×4（保质↑↑）**（@Smokehouse）。  
- **醋渍菜 M**：蔬×3 + 醋×1 + 盐×0.5 → **醋渍菜 ×4（保质↑）**（@Saltery）。  
- **鱼酱 C**：小鱼/内脏×4 + 粗盐×2 → **鱼酱 ×1（调味）**（@Fermentation Jar）。  
- **香料糖浆 R**：糖×1 + 香料×0.5 → **甜香糖浆 ×1（风味+1/宴会+1）**（@Spice & Sugar Bench）。

> **保质（简化建议）**：腌/熏/醋渍 ≫ 面包/馅饼 ≫ 炖菜；若装桶/密封再小幅延长（与酿造系统共享“容器”逻辑）。

## 7）系统收益（对比 DF）
- 从“一体化 Kitchen”到“**灶锅 / 烤炉 / 腌熏**”三条线，**学习成本极低**，却带来：  
  - **更明确的角色**：炖=口粮，烘焙=携带与合餐，腌熏=保质与远征。  
  - **时代风味**：C=鱼酱/橄榄油；M=公共炉+腌熏；R=香料+糖与宴席。  
  - **闭环**：盐（矿业/贸易）、木屑（林业）、脂肪/乳制（畜牧）、醋（酿造）；与既有系统完全耦合。

## 8）实现与平衡建议
1) 将 DF 的 **Kitchen** 拆分为上述工坊；保留“炼脂→牛羊油/猪油”的动作挂到 Hearth & Cauldron。  
2) **吞吐**：Communal Oven 给烘焙 **+20% 批量效率**；腌熏成品 **保质倍数×3**；甜香糖浆/香料盐为**消耗性增益**。  
3) **UI**：建造菜单按“炖 / 烘焙 / 保藏 / 调味”四个按钮组织，鼠标提示直接显示“份数、保质、士气、批量系数”。

## 1) 核心原则
- **时代化增益**：古典→中世纪→文艺复兴分别解锁耕作工具/动力/仓储与压榨效率，形成明确“台阶感”。  
- **土壤肥力**（plot-level，0–3）：  
  - 收获谷物 −1；**休耕** +1（上限 3）；**豆科**收获 +1；**粪肥/堆肥/草木灰**各 +0.5（可叠加，仍受上限 3 限制）。  
- **吞吐倍率**：无动力石臼（Quern）＜手摇磨（过渡）＜**动力磨石**（水/风）。  
- **副产物入链**：秸秆→茅草/垫料/燃料；榨饼→燃料/饲料；酒醋→烹饪/防腐；草木灰→碱液/玻璃/施灰。

## 2) 时代 → 工艺链 → 投入/产出 → 解锁

### CLASSICAL（古典）
**A. 基础耕作（两圃+休耕）**  
- **链路**：整地 → 播种 → 除草 → 收割/脱粒 → 晾晒/入仓  
- **投入**：种子（小麦/大麦/葡萄/橄榄/亚麻/豆科）×S、劳力、工具  
- **产出**：谷物×Y、秸秆×副产、（葡萄→葡萄浆、橄榄→果浆）  
- **解锁**：**田地（C）**、**打谷场/晒场（C）**、**蓄水渠/坎儿井样式灌溉（可选C）**  
- **肥力**：谷物 −1；豆科 +1；休耕 +1；粪肥/草木灰 +0.5

**B. 压榨与葡萄酒（作物→半成品）**  
- **链路**：葡萄 → 踏榨/梁式压榨 → 葡萄汁（must）→（交给酿造）  
- **链路**：橄榄 → 梁式压榨 → 橄榄油 + 榨饼  
- **解锁**：**梁式压榨坊（C）**、**陶罐/安瓿（陶工坊C）**  
- **去向**：油→烹饪/肥皂；榨饼→燃料/饲料

**C. 碾磨（起步）**  
- **链路**：谷物 → **石臼（Quern）** → 粗粉  
- **解锁**：**石臼（C）**  
- **倍率**：吞吐 ×1（基准）

---

### MEDIEVAL（中世纪）
**A. 三圃轮作 & 重犁/马项圈**  
- **链路**：秋粮地 / 春作地 / 休耕地 三分轮换；**重型犁**提升耕深，**马项圈**提升牵引效率  
- **投入**：重型犁（装备）、马项圈（装备）、劳力  
- **产出**：谷物/豆类产量 **×1.15**（轮作开关开启时），播种速度 +25%（装备齐全时）  
- **解锁**：**重型犁（M）**、**马项圈（M）**

**B. 动力磨石（显著台阶）**  
- **链路**：水轮/风车 → 传动 → **磨石**  
- **投入**：谷物×G  
- **产出**：细粉×G、麸皮×副产  
- **解锁**：**水轮（M）/风车（M）**、**磨坊（M）**  
- **倍率**：吞吐 **×2.5**（相对石臼）

**C. 预处理与仓储**  
- **链路**：谷物干燥 / **通风粮仓** / 简易熏硫防蛀（可选）  
- **效果**：季末储损 −50%（建有**通风粮仓（M）**）

**D. 油/酒压榨升级（效率）**  
- **链路**：杠杆/螺杆混合改良 → 出油率提升  
- **解锁**：**改良压榨坊（M）**  
- **倍率**：橄榄→油 **+10%**；葡萄→汁 **+10%**

---

### RENAISSANCE（文艺复兴）
**A. 螺旋压榨 & 梯架葡萄园（效率/品质）**  
- **链路**：**螺旋压榨（Screw Press）**、藤架修剪 → 更稳的出汁/出油  
- **解锁**：**螺旋压榨坊（R）**、**葡萄藤架（R）**  
- **倍率**：在 M 基础上再 **+10%**（总+20% 相对古典梁式）

**B. 精细整籽与风选**  
- **链路**：**簸箕/手摇风箱** → 杂质减少，磨粉品质提升  
- **解锁**：**簸筛台（R）**  
- **效果**：磨粉质量标签 +1（口感/售价挂钩）

**C. 田间改良（仍轻量）**  
- **链路**：**堆肥坑（R）**、行距标杆（半机械播种的前身）  
- **效果**：堆肥 +0.5 肥力；播种失误率微降（+5% 出苗）

## 3) 物品（Items）
**作物/种子**：小麦籽（wheat_seed）、大麦籽、燕麦籽、豆科籽（pea/bean_seed）、葡萄插条、橄榄苗、亚麻籽  
**采收物**：谷物袋（grain_sack）、豆类（pulses）、葡萄（grape_crate）、橄榄（olive_crate）、亚麻梗（flax_stem）  
**中间品**：粗粉（meal_coarse）、面粉（flour_fine）、麸皮（bran）、葡萄汁（must）、酒（wine）、酒醋（vinegar）、橄榄油（olive_oil）、榨饼（presscake）、堆肥（compost）、粪肥（manure）、草木灰（wood_ash）  
**副产**：秸秆（straw）→ 茅草/垫料/燃料

---

## 4) 配方（Recipes，摘要）

### 古典（C）
- **耕作：谷物（两圃）**  
  `输入：wheat_seed×1, 劳力, 工具 → 输出：grain_sack×X, straw×副产`  
  备注：收获后 **肥力 −1**  
- **耕作：豆科（恢复）**  
  `输入：bean_seed×1 → 输出：pulses×X → 收获后 肥力 +1`  
- **休耕（回春）**  
  `输入：地块×1 季 → 输出：肥力 +1（上限 3）`  
- **施肥**  
  `粪肥×1 / 堆肥×1 / 草木灰×1 → 肥力 +0.5（上限 3）`  
- **石臼碾磨（Quern）**  
  `grain_sack×1 → meal_coarse×1`（吞吐×1）  
- **梁式压榨：葡萄/橄榄**  
  `grape_crate×1 → must×1`；`olive_crate×1 → olive_oil×0.8 + presscake×0.2`

### 中世纪（M）
- **三圃轮作（开关）**  
  启用后：谷物/豆类 **产量 ×1.15**；播种速度 +25%（需要**重型犁+马项圈**）  
- **动力磨石（磨坊）**  
  `grain_sack×1 → flour_fine×1 + bran×0.2`（吞吐 **×2.5**）  
- **改良压榨**  
  `olive_crate×1 → olive_oil×0.88 + presscake×0.12`；`grape_crate×1 → must×1.1`  
- **通风粮仓（减损）**  
  `存储：grain_sack` → 季末损耗 −50%

### 文艺复兴（R）
- **螺旋压榨**  
  在 M 基础上 **再 +10% 出率**（总计相对 C **+20%**）  
- **簸筛**  
  `grain_sack×1 → flour_fine(quality+1)`（标签）  
- **堆肥坑**  
  `杂草/厨余/presscake×1 → compost×1` → 田地使用：肥力 +0.5

## 5) 建筑（Workshops / Areas）
- **田地（C）**：耕作基础单元  
- **打谷场/晒场（C）**：脱粒/晾晒  
- **石臼（C）**：入门碾磨  
- **梁式压榨坊（C）**：葡萄/橄榄初榨  
- **灌溉渠（C，可选）**：旱地减损  
- **重型犁（M，装备）**、**马项圈（M，装备）**  
- **水轮/风车 + 磨坊（M）**：动力磨石  
- **通风粮仓（M）**：减损  
- **改良压榨坊（M）**：压榨效率+10%  
- **螺旋压榨坊（R）**：在 M 基础再+10%  
- **簸筛台（R）**：磨粉质量+1  
- **堆肥坑（R）**：长期土壤回春

---

## 6) 平衡默认值（可直接入表）
- **轮作倍率**：两圃=1.00；三圃=**1.15**  
- **磨粉吞吐**：石臼 ×1；动力磨石 **×2.5**  
- **压榨效率**：C 基线；M **+10%**；R **+20%**（相对C）  
- **肥力上限**：3；**休耕 +1**、豆科 +1、粪肥/堆肥/草木灰 +0.5  
- **仓储损耗**：有通风粮仓 −50%

---

## 7) 与其他系统的挂接
- **酿造**：`must → wine`；`barley → malt → beer`（酿造配方在轻工业侧实现）  
- **烹饪/肥皂**：`olive_oil` 进入烹饪与皂化；`vinegar` 作为调味/防腐  
- **玻璃/化学**：`wood_ash → lye/potash`；继续保留你的玻璃与化学链  
- **纺织**：`flax_stem →（后续）亚麻纤维`（在纺织系统实现）

## 8) 数据字段建议（便于直接落库）
- **fields.csv**：`plot_id, era, fertility(0-3), rotation(two/three), fallow(bool), irrigation(bool)`  
- **crops.csv**：`crop_id, seed, era, base_yield, fertility_cost, byproducts`  
- **recipes_agri.csv**：`id, era, inputs(json), outputs(json), time, building, multiplier_tag`  
- **buildings_agri.csv**：`id, era, throughput, power_req, storage_bonus, notes`

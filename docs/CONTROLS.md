# Controls

- WASD: Move camera
- Q / E: Change Z level
- Mouse move: Move cursor over tiles
- Left-click on tile: Open Tile Info panel
- Tile Info panel shows the clicked tile and layers z=0..7 (ESC closes)
- Left-click on bottom-left dock [F1..F7]: Open panels
- Left-click on bottom-center [Z][X][C]: Open quick menus
- F9: Cycle navigation overlays
- ESC: Close/back; if none open, toggles Pause overlay
- Right-click: Cancel/close current UI
- Ctrl + Mouse Wheel: Zoom in/out (1x–4x)

Debug Menu
- Toggle with F12 or Backquote (`) on some keyboards
- Semi-transparent centered panel showing status
- Click “Spawn Dwarf” to add a debug marker at cursor

Notes
- Camera and cursor are independent. Camera moves with WASD; cursor follows the mouse.
- When zoomed, each world tile is rendered as a block of cells. Cursor snapping still targets the underlying world tile.
